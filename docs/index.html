Get Started 
local texture = 0
local angle = 0

function g_init()
    texture = loadTexture("gbox.png")
    title("GBox Demo")
end

function g_loop()
    fill(1, 0.8, 1)
    
    useTexture(texture)
    
    beginDraw(DRAW_QUADS)
        color(0.7, 0.0, 0.8)
        vertex(0.5, -0.5, 0.0)
        texCoord(0, 0)

        vertex(0.5, 0.5, 0.0)
        texCoord(1, 0)

        color(0.0, 0.0, 0.8)
        vertex(-0.5, 0.5, 0.0)
        texCoord(1, 1)

        vertex(-0.5, -0.5, 0.0)
        texCoord(0, 1)
    endDraw()

    if btn(K_SPACE) then
        angle = angle+0.01
        camera(
            math.sin(angle)*4, 0, math.cos(angle)*4,
            0,0,0
        )
    end
end

Introduction

What is GBox?

GBox is a lightweight, high-performance game engine built with C++, OpenGL, and SDL3, designed for developers who want to create games with Lua scripting. With its immediate-mode rendering style and simple API, GBox makes 2D/3D game development accessible while providing low-level control for advanced users.

Why Choose GBox?

GBox offers a unique combination of C++ performance with Lua's flexibility. Unlike heavier engines, GBox provides a minimal abstraction layer that gives you direct control over rendering while handling the complex backend work.

Key Features

High Performance

Optimized rendering pipeline and efficient memory management for buttery smooth gameplay.

Component System

Flexible ECS architecture that makes your game code modular and maintainable.

Built-in Physics

Powerful physics engine with collision detection, rigid bodies, and force management.

Install GBox Engine

Windows

  1. Download GBox-Windows-x64.exe
  2. Run the installer executable
  3. Check "Add to PATH" during installation
  4. Open Command Prompt and verify: 
    gbox --version
Download for Windows

Linux

# Download and install
wget [LINUX_HERE]
tar -xzf linux.zip
sudo mv gbox /usr/local/bin/

# Verify installation
gbox --version

Quick Start Guide

1. Create a Lua script

function g_init()
    title("My GBox Game")
end

function g_loop()
    fill(0.1, 0.1, 0.1)
end

2. Run your game

gbox mygame.lua

For debugging:

gbox -debug mygame.lua
